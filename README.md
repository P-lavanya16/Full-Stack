# Patient Documents Portal

A full-stack application that allows patients to upload, view, download, and delete medical PDF documents.

## Features
- Upload PDF documents
- View list of uploaded files
- Download files
- Delete files
- Backend API with Express & MongoDB
- Frontend built with React
